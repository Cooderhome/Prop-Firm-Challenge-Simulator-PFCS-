# Prop-Firm-Challenge-Simulator-PFCS-
PFCS a is trading tool that mimics 2-step prop firm evaluations, enforcing rules like max daily/overall drawdown and per-trade loss, tracks equity, PnL, and progress, provides instant feedback on violations, visualizes performance with charts, and logs detailed trade data via a modern Glassmorphism UI using Python, Flask, SQLite, and Chart.js
